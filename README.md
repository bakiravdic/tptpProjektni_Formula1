# tptpProjektni_Formula1

## O projektu
Ovaj projekat predstavlja interaktivnu web stranicu posvećenu Formuli 1, razvijenu u svrhu ispunjavanja akademskih zahtjeva. Stranica nudi dinamički pregled F1 timova kroz interaktivnu pretragu, simulator bodovanja trke, te sekcije za prijavu na newsletter i simulaciju prijave tehničkih grešaka.

## Tehnologije
- HTML5 (Struktura i Image Map koncepti)
- CSS3 
- JavaScript (Manipulacija DOM-om, ručna validacija formi i event listeneri, bez biblioteka)

## AI alati koristeni u projektu
- Gemini: Proračun tačnih koordinata za interaktivne zone na slici volana, optimizacija JavaScript koda.

## Napomene
Audio snimak preuzet je sa stranice ZEDGE i ujedno je besplatan.
Sve slike koje su koristene u projektu su besplatno preuzete sa stranica CLEANPNG,HICLIPART,F1 FANART.
